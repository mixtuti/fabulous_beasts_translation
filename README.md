# 中日翻訳カード v6

## 追加内容

- `input.html` を追加
- 中国語入力補助をメインページから別ページへ移動
- 頻出100字のクリック入力パネルを追加
- 文字 / pinyin / 意味で検索可能
- よく使う語句・記号パネルを追加
- 音声読み上げ・コピー・クリア機能つき

## 使い方

- `index.html`: カード一覧・検索・音声再生
- `input.html`: 中国語入力補助ページ

## 頻出100字について

`input.js` の `COMMON_CHARS` に登録しています。
必要に応じて追加・削除・意味の調整ができます。


## v7 追加: カード入力補助

- `card-input.html` を追加
- フォームから `data.json` 用のカードJSONを生成
- 既存の `data.json` を読み込んで次のIDを自動入力
- 登録済み単語から `contains` をクリック選択
- 未登録語も手入力で `contains` に追加可能
- カードJSONをコピー、または配列末尾用にカンマ付きコピー
